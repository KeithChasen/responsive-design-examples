PHP DEVELOPMENT

PHP is a modern programming language and one of the most popular languages these days to be used for writing server side applications, REST APIs etc. It has huge community, quite good documentation and a lot of nice frameworks e.g. Laravel and Symfony.

FRONTEND DEVELOPMENT

It's impossible even to imagine modern web development without such tools as HTML5, CSS3 and JavaScript. The latest version of HTML provide us with new semantic tags such as header, footer, article, nav and many others which help us not to only write more cleaner code but also help search engines index our website much more faster. With new version of HTML we are now allowed to add video and audio content to our websites without flash. We can even create games in there! New version of CSS gave us a lot wonderful possibilities to make our websites look beautiful, media queries gave us easy to use and flexible approach to make our websites behave responsibly on any kind of device either it's gonna be desktop or laptop, tablet or mobile phone. The last but not least is the JavaScript. Language that initially was created as a simple tool for making static web pages more interactive. Nowadays we can't find a single website which is not using JS on frontend. These days JS grown into a huge ecosystem with its realization for backend development (NodeJS), package manager (NPM) and a lot of framework for SPA (Single Page Application) such as Angular, Vue and React (Alright, Alright, it's a library, don't punch me)
